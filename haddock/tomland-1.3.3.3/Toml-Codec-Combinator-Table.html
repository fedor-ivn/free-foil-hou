Description

Contains TOML-specific combinators for converting between TOML and user data types.

Tables can be represented in TOML in one of the following ways:

foo =
    { x = ...
    , y = ...
    , ...
    }

Or

[foo]
    x = ...
    y = ...
    ...

Since: 1.3.0.0
